About

Uddhav Shrimali.

Full-stack developer, WordPress specialist, and builder of clean web systems. I write code that solves real problems — no bloat, no shortcuts.

Background

Built from the ground up.

Started coding out of curiosity — tearing apart WordPress themes to understand how they worked. That curiosity became a career. I've shipped everything from simple landing pages to complex multi-vendor platforms and custom REST API systems.

I operate as an independent contractor. That means one person writes, reviews, and ships your entire codebase. No handoffs. No communication gaps. No agency markup.

My specialisation is the WordPress ecosystem — but I can build and connect anything that runs on a server or in a browser.

Engineering Philosophy

// 01

Clean code over clever code.

Code you can read six months later is more valuable than code that shows off. I write for maintainability first.

// 02

Performance is not optional.

Every database query, every asset load, every API call is a decision. I build with page weight and response time in mind from the start.

// 03

Ship, then improve.

Working software in production is worth more than perfect architecture in a planning doc. I bias towards delivery.
